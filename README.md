## Setup

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/roela12/myportfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd myportfolio
   ```
3. Install node and run the following command:
   ```bash
   npm run dev
   ```
